# Page Override: reports

## Purpose
Present discussion outcomes and analytics as evidence, not decoration.

## UI Rules
- Prioritize neutral palettes with semantic color only for meaning.
- Keep metric cards structurally consistent across summary, participation, and scoring.
- Preserve readability in dense analytical sections and tables.

## Interaction Rules
- Generation and refresh actions should be clear and stable.
- Tabs should expose role-appropriate depth without visual clutter.
